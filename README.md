## 使用方法
1. 下載本程式，你必須有```cURL```
2. 設定```config.php```
 * ```cURL```為執行命令，可使用Chrome主控台，打開`Network`頁籤，在`login.php`上按右鍵，點選`Copy`>`Copy as cURL (bash)`
 * ```times```為執行次數
 * ```sleep```為發送兩次請求之間的暫停時間(毫秒)
1. 執行```index.php```
